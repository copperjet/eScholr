## CARD 2 — Classroom Teacher (HRT)

**Your role:** Take daily attendance, enter marks, write report comments.

### First Login
1. Open **eScholr** → enter school code **`LOTS`**
2. Enter your email + password → tap **Sign In**

### Daily Tasks
| Task | Where |
|---|---|
| Take attendance | Home → **Attendance** → select date → mark each student → **Submit Register** |
| View attendance history | **Attendance** → **History** tab |
| Enter marks | **Marks** → select assessment (FA1, FA2, Summative) → enter score per student |
| Write report comment | **Reports** → tap student → edit comment → **Submit for Approval** |
| Post announcement | **Announcements** → ＋ |

### Key Rules
- Attendance must be submitted before **9:30 AM** each day
- Marks can be edited until the admin closes the marks window
- Report comments must be submitted **before** the admin approves

### If You Make a Mistake
- **Attendance:** An admin can unlock and correct it — contact your admin
- **Marks:** Edit the mark directly — changes are logged automatically

## CARD 3 — Subject Teacher (ST)

**Your role:** Enter marks for your subjects across streams.

### First Login
1. Open **eScholr** → school code **`LOTS`** → sign in

### Your Main Screen
Home shows your assigned streams and subjects.

| Task | Where |
|---|---|
| Enter marks | **Marks Entry** → select stream → select subject → enter scores |
| View mark progress | **Marks Entry** → progress bar at top shows % complete |
| View students | **Students** tab → browse your stream |

### Key Rules
- Enter marks as a **percentage (0–100)**
- FA1 = 20%, FA2 = 20%, Summative = 60% — the app calculates grades automatically
- IGCSE / A-Level streams: Summative = 100%

## CARD 4 — Finance Officer

**Your role:** Record payments, generate receipts, view finance reports.

### First Login
1. Open **eScholr** → school code **`LOTS`** → sign in

### Daily Tasks
| Task | Where |
|---|---|
| Record a payment | **Finance** → search student → **Record Payment** → enter amount + method |

## CARD 5 — Parent

**Your role:** View your child's reports, marks, homework, and announcements.

### First Login
1. Download **eScholr** from the App Store / Google Play
2. Enter school code **`LOTS`**
3. Enter your email + password (provided by the school office)

### What You Can See
| Section | What's there |
|---|---|
| **Home** | Your child's name, class, latest announcement |
| **Reports** | Released report cards — tap to view PDF |
| **Homework** | Assignments set by teachers |
| **Announcements** | School-wide and class notices |
| **Messages** | Direct messages from teachers |

### Push Notifications
You'll receive a notification when:
- A report card is released ✅
- A new announcement is posted 📢
- A teacher sends you a message 💬

> **Privacy:** You can only see your own child's data.

## CARD 6 — Platform Admin

**Your role:** Manage all schools on the eScholr platform.

### Access
Web or mobile → sign in with platform admin credentials (no school code needed)

### Key Tasks
| Task | Where |
|---|---|
| Add a new school | **Schools** → ＋ → fill school profile → **Create** |
| Manage school subscription | **Schools** → select school → **Subscription** |
| View all users | **Users** tab → filter by school or role |
| Export school data | **Schools** → select school → **Export Data** |
| Send platform announcement | **Announcements** → audience: *All Schools* |

### Monitoring
- Check **Audit Log** for suspicious activity
- Check **Notification Log** for push delivery failures
- Database health: Supabase Dashboard → **Reports**
